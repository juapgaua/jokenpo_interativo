# Jokenpô – Projeto Interativo em JavaScript

Este é um jogo de **Pedra, Papel e Tesoura** desenvolvido para a disciplina **Desenvolvimento em JavaScript**.  
Possui animações, lógica completa, interface moderna, responsiva e interativa.

## Funcionalidades

- Escolha entre **Pedra, Papel ou Tesoura**
- Destaque visual da escolha do jogador e do computador
- Resultado totalmente dinâmico:
  - **Você ganhou**
  - **Você perdeu**
  - **Deu empate**
- Placar automático (Vitórias, Derrotas e Empates)
- Botão de reiniciar partida
- Layout responsivo para celular, tablet e desktop
- Footer institucional

## Tecnologias Utilizadas

- **HTML5**
- **CSS3** (incluindo animações e responsividade)
- **JavaScript** (lógica e interatividade)
